# Very Interesting Videos — Automation

Narzędzie automatyzujące przygotowanie i przesyłanie krótkich filmów na YouTube.

- Repozytorium: https://github.com/TwojLogin/NazwaProjektu (zastąp własnym linkiem)
- Kontakt: veryinterestingvideos804@gmail.com

## Co robi ta aplikacja

Automation generuje skrypt, dobiera darmowe zasoby (Pexels/Pixabay), syntetyzuje lektora (offline: pyttsx3), składa wideo i — po autoryzacji — przesyła je na Twoje konto YouTube.

## Polityka prywatności

Pełna polityka prywatności dostępna tutaj: [PRIVACY.md](PRIVACY.md)

## Jak uruchomić (szybki start)

1. Skopiuj repozytorium lokalnie.
2. Utwórz plik `.env` z wymaganymi kluczami (GROQ_API_KEY, PEXELS_API_KEY itd.).
3. Uruchom:

```bash
python main.py
```

Zobacz `README` w repozytorium po szczegóły konfiguracji.
